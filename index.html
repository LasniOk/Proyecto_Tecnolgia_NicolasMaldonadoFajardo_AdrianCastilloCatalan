    Grupo 1: Sociedad Digital y Datos

    La Sociedad Digital

    La sociedad digital es un entorno en el que las tecnologías de la información y la comunicación (TIC) son fundamentales para el desarrollo de actividades cotidianas. Esto incluye la forma en que las personas se comunican, trabajan, acceden a la educación, consumen productos y servicios, y participan en la vida social y política. La digitalización ha transformado industrias enteras, ha creado nuevas formas de empleo y ha facilitado el acceso a la información, contribuyendo a una mayor globalización e interconexión entre las personas y los países.

    ¿Qué es un dato?

    Un dato es una representación simbólica de una realidad o un hecho, que puede ser procesado para obtener información útil. En el ámbito digital, los datos se pueden almacenar, compartir y analizar para distintos propósitos.

    Piramide DIKW

    La pirámide DIKW representa una jerarquía que muestra la relación entre datos, información, conocimiento y sabiduría:

    • Datos: Hechos sin procesar o elementos brutos (números, símbolos).
    • Información: Datos procesados o contextualizados para tener significado.
    • Conocimiento: Interpretación y análisis de la información para generar comprensión.
    • Sabiduría: Uso del conocimiento de manera reflexiva y ética para tomar decisiones acertadas.

    Grupo 2: Placa Base y Software

    Componentes de una Placa Base

    La placa base, o motherboard, es el componente principal de un dispositivo que conecta y permite la comunicación entre todos los demás componentes, como el procesador, la memoria, y los dispositivos de almacenamiento.

    Una placa base incluye componentes clave como el procesador (CPU), la memoria RAM, el chipset, las ranuras de expansión, y los puertos de entrada y salida, que permiten que todos los componentes del sistema trabajen juntos.

    Los componentes de una placa base son:

    • Socket del procesador (CPU)
    • Ranuras de memoria RAM
    • Chipset
    • Ranuras de expansión (PCIe, PCI)
    • Conectores de almacenamiento (SATA, M.2)
    • Conectores de energía (24 pines, CPU 8 pines)
    • BIOS/UEFI
    • Puertos USB
    • Conectores de audio y red
    • Batería CMOS
    • Conectores para ventiladores

    ¿Qué es el Software?

    El software es el conjunto de programas y aplicaciones que permiten la operación de un sistema informático. Se divide en software de sistema, como sistemas operativos, y software de aplicación, como los programas que usamos a diario.

    Software de aplicacion

    El software de aplicación es un tipo de programa diseñado para realizar tareas específicas para el usuario. Estos programas ayudan a ejecutar diversas aplicaciones que permiten realizar trabajos, gestionar datos o entretenerse.

    Ejemplos:

    • Procesadores de texto (como Microsoft Word, Google Docs).
    • Hojas de cálculo (como Microsoft Excel, Google Sheets).
    • Navegadores web (como Google Chrome, Mozilla Firefox).
    • Software de edición de imágenes (como Adobe Photoshop, GIMP).

    Software de Sistema

    El software de sistema es el conjunto de programas que gestionan y controlan el hardware del ordenador y proporcionan una plataforma para que el software de aplicación funcione. Actúa como intermediario entre el hardware y el software de aplicación.

    • Sistemas operativos (como Windows, macOS, Linux).
    • Controladores de dispositivo (drivers) que permiten que el sistema operativo se comunique con hardware específico.
    • Utilidades del sistema que realizan tareas de mantenimiento y optimización (como programas de desfragmentación o antivirus).

    Licencias de software

    Las licencias de software son acuerdos legales que permiten a los usuarios utilizar un software bajo ciertas condiciones establecidas por el desarrollador o distribuidor. Estas licencias determinan cómo se puede usar, distribuir, modificar o compartir el software.

    Tipos de licencia de software

    • Licencia Proprietaria: El usuario paga por el software y no tiene derecho a modificarlo o distribuirlo. Ejemplos incluyen Windows y Adobe Photoshop.
    • Licencia de Código Abierto: Permite a los usuarios ver, modificar y redistribuir el código fuente. Ejemplos incluyen las licencias MIT y GPL.

    Grupo 3: Redes

    Las Redes

    Una red es un conjunto de dispositivos interconectados que comparten recursos e información, como computadoras, servidores y dispositivos móviles, a través de medios físicos o inalámbricos.

    Las redes permiten la interconexión de dispositivos para compartir recursos e información. Pueden ser redes locales (LAN), redes de área amplia (WAN), o incluso redes globales como Internet.

    Tipos de red

    • Red de área personal (PAN): Conecta dispositivos dentro de un rango muy cercano, generalmente hasta 10 metros. Se usa para sincronizar dispositivos personales, como teléfonos inteligentes, tabletas y computadoras portátiles. Permite compartir archivos, datos y recursos como impresoras o conexiones a internet de manera eficiente y rápida.
    • Red de área local (LAN): Conecta dispositivos dentro de un área geográfica limitada, como un hogar, una oficina o un campus universitario. Facilita el intercambio de archivos, el acceso a recursos compartidos (como impresoras y servidores) y la comunicación interna entre dispositivos. Es fundamental para redes corporativas y redes domésticas.
    • Red de área metropolitana (MAN): Cubre áreas más amplias que una LAN, como ciudades o áreas metropolitanas. Permite la interconexión de múltiples redes LAN y facilita el acceso a servicios de internet y recursos compartidos en una región extensa. Las MANs son utilizadas por proveedores de servicios para ofrecer conectividad en áreas urbanas.
    • Red de área amplia (WAN): Se extiende a través de grandes distancias geográficas, conectando redes LAN y MAN ubicadas en diferentes regiones, países o incluso continentes. Es esencial para la comunicación global y el intercambio de datos entre sucursales de empresas multinacionales. Los proveedores de servicios de internet y las empresas utilizan WANs para ofrecer conectividad a largo alcance.
    • Red de área de almacenamiento (SAN): Diseñada específicamente para la conexión de dispositivos de almacenamiento a servidores, optimizando el acceso y la gestión de datos en grandes volúmenes. Se utiliza en entornos empresariales para mejorar el rendimiento de las aplicaciones, la disponibilidad de datos y la recuperación ante desastres. Permite a las organizaciones gestionar grandes cantidades de datos de manera eficiente.

    Grupo 4: Inteligencia Artificial

    Breve Historia de la Inteligencia Artificial

    La inteligencia artificial (IA) comenzó en la década de 1950, cuando los primeros científicos, como Alan Turing y John McCarthy, plantearon la posibilidad de que las máquinas pudieran realizar tareas que requieren inteligencia humana. Desde entonces, la IA ha evolucionado rápidamente, influenciada por avances en el aprendizaje automático, la capacidad computacional y el almacenamiento de datos.

    • Vehículos autónomos
    • Procesamiento del lenguaje natural (NLP)
